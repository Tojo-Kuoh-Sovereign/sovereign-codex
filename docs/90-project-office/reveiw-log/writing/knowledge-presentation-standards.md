# Knowledge Presentation Standards

---

## Purpose

Translate the Knowledge Presentation Principles into practical documentation standards.

This document defines *how* knowledge should be presented throughout the Sovereign Codex.

Unlike the Principles document, these standards may evolve as the platform grows and should reflect current implementation best practices.

---

## Relationship to the Principles

Every standard should be supported by one or more Knowledge Presentation Principles.

Standards should never contradict the philosophical intent established by the Principles document.
