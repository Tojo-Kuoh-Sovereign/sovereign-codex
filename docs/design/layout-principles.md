Interaction Principles

Knowledge First

The interface should never compete with the content.

Deliberate Interaction

Every interaction must have a clear purpose.

Reduce Cognitive Load

Users should spend their mental effort understanding the subject, not navigating the interface.

Reveal, Don't Distract

Progressive disclosure is preferred over overwhelming users with information.

Respect User Agency

Provide trusted information and supporting evidence.
Allow users to draw their own conclusions.

Confidence Through Clarity

Users should feel confident because information is organised, sourced and well-presented—not because the interface is persuasive.
