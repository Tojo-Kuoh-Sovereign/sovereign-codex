# Project Philosophy

Sovereign Codex exists to make reliable information easier to find, easier to understand, and easier to maintain.

The project was founded on the belief that good documentation is more than simply collecting information. Knowledge should be organised, verified, and presented in a way that allows people to quickly find answers while understanding the context behind them.

Rather than creating another collection of guides, Sovereign Codex aims to become a structured knowledge base that can evolve alongside Black Desert Online.

---

## Core Principles

The project is guided by a small number of fundamental principles.

### Accuracy over speed

Information should be verified before it is published.

Providing reliable documentation is more valuable than publishing information first.

### Structure over volume

Well-organised documentation is more useful than a large collection of disconnected pages.

Every page should have a clear purpose and a logical place within the knowledge base.

### Clarity over complexity

Technical subjects should be explained as simply as possible without sacrificing accuracy.

Documentation should be approachable for new players while remaining valuable to experienced players.

### Continuous improvement

Documentation is never considered "finished."

As new information becomes available or game systems change, the Codex should continue to evolve.

### Community driven

While the Codex is curated to maintain consistency and quality, community feedback, corrections, and discussion are an important part of keeping the knowledge base accurate and relevant.

---

## Our Goal

The objective of Sovereign Codex is not to document everything.

The objective is to document things well.

Every page should contribute towards building a trusted reference that players can confidently return to whenever they need reliable information.
