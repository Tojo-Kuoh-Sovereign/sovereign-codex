# Colour Palette

## Philosophy

Sovereign Codex does not begin with colours.

It begins with materials.

Every colour used throughout the platform represents a physical material chosen for its symbolism, permanence and craftsmanship.

Rather than selecting colours based solely on aesthetics, the palette is derived from materials that embody the values of Sovereign Codex.

The digital interface becomes a translation of these materials into light.

## Material Palette

| Material     | Represents             | UI Usage                 |
| ------------ | ---------------------- | ------------------------ |
| Obsidian     | Foundation, permanence | Backgrounds              |
| Forged Steel | Structure, engineering | Layout, borders          |
| Aged Gold    | Stewardship, prestige  | Highlights, metadata     |
| Sapphire     | Verified knowledge     | Links, active navigation |
| Parchment    | Human knowledge        | Documentation surfaces   |
| Graphite     | Neutral hierarchy      | Secondary text           |
